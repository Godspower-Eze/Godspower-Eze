## Builds

### 🔐 Cryptography & Zero-Knowledge

- 🛡️ **[verifiers-toolkit](https://github.com/Godspower-Eze/verifiers-toolkit)** — Circom/Noir circuits to Cairo Groth16/UltraHonk verifiers on StarkNet in minutes
- 🦀 **[zk-lang](https://github.com/Godspower-Eze/zk-lang)** — an experimental ZK circuit language
- 🔮 **[fhe-and-lbc](https://github.com/Godspower-Eze/fhe-and-lbc)** — implementations from Fully Homomorphic Encryption (FHE) and Lattice-Based Cryptography
- 🔒 **[pqc-ml-kem.rs](https://github.com/Godspower-Eze/pqc-ml-kem.rs)** — Rust library for ML-KEM (Module-Lattice-Based Key-Encapsulation Mechanism)
- 🕸️ **[modcc.xyz](https://github.com/Godspower-Eze/modcc.xyz)** — web tool for the sumcheck protocol; univariate and multilinear Lagrange interpolation over finite fields
- ⚙️ **[modcc](https://github.com/Godspower-Eze/modcc)** — Rust library for univariate, multivariate, and multilinear polynomial operations over finite fields; API backend for modcc.xyz
